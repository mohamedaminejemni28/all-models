# Metrics

CV accuracy is the cross-validation accuracy. Balanced accuracy is preferred when class distributions may be imbalanced because it accounts for both classes.

Test accuracy is performance on the held-out test set.

Sensitivity measures how well the positive class is detected. Specificity measures how well the negative class is detected.

F1 score balances precision and recall. ROC-AUC evaluates ranking quality across thresholds.

A strong model should have high average performance, low cross-validation variance, balanced sensitivity and specificity, and interpretable important features.
