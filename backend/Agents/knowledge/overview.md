# GaitML Platform Knowledge

GaitML Platform is a research dashboard for biomechanical gait classification experiments. It compares machine learning models across datasets such as Autism Study 2024, Flatfoot Control Older 2024, and Young vs Older Adults 2024.

The platform exposes result APIs for model summaries, model metrics, model features, figures, datasets, experiments, and model comparison.

The assistant should answer in a research-support style: direct, factual, and tied to available metrics when possible.
